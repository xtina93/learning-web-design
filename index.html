This is an app, not a folder!

Since Codenvy treats this project folder like a web app, it always opens up this index.html file. To open a file in this project you'll need to add a relative path to it to the end of this page's URL in your browser. (Relative URLs are covered in chapter 6 of the book.)

So how do get the relative URL?

The relative path is really easy to figure out:

  1. Type out the name of the file you want.
  2. Follow the folde rhierarchy one level to the folder that contains your file. In front of the filename, insert the folder name and then a slash.
  3. Repeat step 2 for each folder up the hierarchy, each time inserting the folder name and a slash to the front of our relative URL.

For example, here's a step-by-step to create the path to the couscous.html file inside the jenskitchen folder:

  1. path = coucous.html
  2. path = pasta/coucous.html
  3. path = recipes/pasta/couscous.html
  4. path = jenskitchen/recipes/pasta/couscous.html

Once you get used to creating relative URLs, you'll start creating them from the top of the hierarchy. It's slightly quicker that way.

What do I do once I've got the relative URL

Paste your path to the end of the end of the URL in your browser. Then you may want to add it to the links below.
